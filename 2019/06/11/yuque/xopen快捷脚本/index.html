xopen快捷脚本

Published on Jun 11, 2019 in iOS
本文字数：106 字 | 阅读时长 ≈ 1 min

教程

自定义xopen快捷脚本，在终端中快速打开项目

详细教程

  1. 创建 xopen 文件
  2. 编辑 xopen 内容
#!/usr/bin/env ruby
require 'shellwords'

proj = Dir['*.xcworkspace'].first
proj = Dir['*.xcodeproj'].first unless proj

if proj
   puts "Opening  #{proj}"
   `open #{proj}`
else
  puts "No  xcworkspace|xcproj  file  found"
end
  1. 将 xopen 文件移入 /usr/local/bin 目录下
  2. 添加权限 chmod 777 xopen
  3. 在终端中，cd到项目目录下，执行xopen
