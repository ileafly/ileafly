使用语雀管理博客

Published on Nov 09, 2018 in Hexo
本文字数：398 字 | 阅读时长 ≈ 1 min

教程

之前一直使用hexo搭建自己的个人博客，用的久了就发现随着博客数量的增加，管理越来越麻烦，之前也尝试过使用hexo-admin，hexo-admin虽然能够达到管理博客的目的，不过界面有点丑，不能忍，一直想找一个替代方案，最近在语雀中读到了一篇博客静态博客使用语雀编辑器，文中介绍了如何利用语雀管理博客，决定一试。

Hexo搭建博客

首先需要先利用hexo搭建博客，具体的搭建流程可以参考Github+Hexo搭建个人博客详细教程

安装语雀插件

yuque-hexo是一个Node.js环境下的语雀下载器。

  1. 安装yuque-hexo

  2. 注册语雀，创建知识库，公开知识库，获得博客知识库的地址，比如我的语雀知识库地址

  3. 在Hexo博客的目录下面找到package.json文件，进入如下配置：

{
"name": "hexo-blog", // 这里一般原来就有，直接添加下面的内容即可
"yuqueConfig": {
    "baseUrl": "https://www.yuque.com/api/v2",
    "login": "leafly",
    "repo": "blog",
    "postPath": "source/_posts/yuque"
  },
}
  1. 同步文章
$ yuque-hexo sync

执行完毕，会自动将语雀上的文章同步到source/_post/yuque文件夹下

$ hexo g
$ hexo d

将博客内容同步到github。
